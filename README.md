# Camp RC Car
## Code for campers to use when creating their RC cars.

The `Dashboard` class starts a SoftAP wifi network and hosts a dashboard on it. Buttons can be added with actions that the code will take when buttons are pressed and released.

The `Car` class is a simple interface to drive a pair of motors on a L289N. It includes easy-to-read function names such as `driveForward()`.
